# 🚀 Next Level Week!
![Capa do projeto]()

## 🎯 Objetivo
Criação de uma aplicação WEB para que seja possível encontrar pessoas para jogar algum jogo junto.

## ⚙️ Funcionalidades
- Procurar por alguém para formar dupla em jogos
- Criar um anúncio para que alguém ache você
